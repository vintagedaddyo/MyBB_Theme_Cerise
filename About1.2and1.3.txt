Where is there a 1.2? and 1.3? ( ie: Cerise 1.2 & Cerise)

Cerise 1.2-theme.xml  -> 1.2
Cerise-theme.xml -> 1.3

You will find that in this main directory we include the latest xml file of theme, ie: Cerise-theme.xml, but, inside the documentations folder you will find directories ie: 1.2 and 1.3 and inside each you will find directories for example like: 1.8.39, ok, this is because I opted to include the older variant of the theme ie: 1.2 and then obviously provide the current version ie: 1.3 .xml files just in case the end user decided they wanted to use either or as their base to then use and improve upon. Hopefully this explanation makes sense. Best of luck! :)
